# Final Year Project Website

A modern website built with Next.js, TypeScript, Tailwind CSS, and ESLint.

## Getting Started

### Prerequisites

- **pnpm** (required for this project)
- Node.js 18+

### Installation

First, install the dependencies:

```bash
pnpm install
```

### Development

Run the development server:

```bash
pnpm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Production

Build the application:

```bash
pnpm run build
```

Start the production server:

```bash
pnpm run start
```

### Linting

Check code quality:

```bash
pnpm run lint
```

## Project Structure
